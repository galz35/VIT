# Reglas de Interacción para Antigravity

## Estilo de Comunicación
- **Idioma**: ESPAÑOL (siempre).
- **Tono**: Directo, simple y eficiente. Sin adornos ni halagos innecesarios.
- **Prioridad**: "Simple + Eficiente + Funciona + Bonito". Gana siempre.

## Desarrollo (React / Frontend)
- **Aesthetic**: Diseños premium, modernos y limpios.
- **Eficiencia**: Si algo ya funciona en la PWA de React, no inventar de cero a menos que sea para optimizar rendimiento real.
- **Skills**: Usar `verify_hierarchy` para asegurar que los permisos corporativos nunca se rompan con lógica "hardcoded".

## Flujo de Trabajo
1. Analizar el problema técnicamente.
2. Proponer la solución más simple que cumpla con el estándar visual "bonito".
3. Implementar y verificar.
